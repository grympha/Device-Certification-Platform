# Windows Agent

Windows Agent is planned for Phase 2 after the Android MVP is validated.

The future PowerShell script will check:

- Windows version
- CPU
- RAM
- Available storage
- Display resolution
- Internet connectivity
- LMX Content installed status
- LMX Content version
- LMX Content running status
- Startup/autologin readiness
- Cache/log folder existence
- JSON report generation
- Upload to backend API
